# HelpLink Mobile Application

HelpLink is an Android mobile application designed to support neighbourhood assistance. The app allows users to request help and offer assistance within their local community.

## Features
- Request help from nearby community members
- Offer assistance to others
- Simple and user-friendly interface
- Community support system

## Technologies Used
- Kotlin
- Android Studio
- XML Layouts

## Project Purpose
This project was developed as part of a university mobile application development project to demonstrate Android development skills.
